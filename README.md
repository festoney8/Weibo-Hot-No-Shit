## 功能

- 净化微博热搜边栏，展示全部热搜，去除闲杂内容（娱乐生活类），支持按照关键词/正则过滤热搜

- 需登录微博后使用

- 微博热搜就是一大坨，本插件只会让它变成一小坨，聊胜于无
